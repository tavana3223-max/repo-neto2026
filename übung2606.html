CSS-Grundlagen in der Praxis

Diese kleine Seite zeigt Selektoren, Farben und Textgestaltung.

1. Selektoren

Ein Element-Selektor spricht alle passenden HTML-Elemente an.

Eine Klasse kann für mehrere Elemente wiederverwendet werden.

Eine ID wird nur für ein eindeutiges Element verwendet.

2. Farben

Farbnamen sind leicht zu lesen.

Hex-Werte sind sehr verbreitet.

RGB-Werte zeigen das Farbmodell klar.

3. Lesbarer Text

Gute Lesbarkeit entsteht durch passende Schriftgröße, ausreichend Zeilenhöhe und einen klaren Kontrast.
